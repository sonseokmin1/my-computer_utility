# 내 컴퓨터 설치 프로그램 목록 조회기 매뉴얼

## 1. 프로그램 개요
이 프로그램은 현재 컴퓨터에 설치된 모든 프로그램의 목록을 터미널(콘솔) 창에 빠르고 간편하게 출력해 주는 도구입니다. 복잡한 설정 없이 실행 파일 하나만으로 바로 확인할 수 있어, 컴퓨터 상태를 점검하거나 불필요한 프로그램을 정리할 때 유용하게 사용할 수 있습니다.

## 2. 주요 기능
- **설치 프로그램 목록 출력**: Windows 레지스트리(HKEY_LOCAL_MACHINE, HKEY_CURRENT_USER)를 검색하여 설치된 모든 소프트웨어(32비트 및 64비트)의 이름을 추출하고 정렬하여 보여줍니다.
- **설치된 프로그램 개수 안내**: 현재 설치된 프로그램의 총 개수를 하단에 표시해 줍니다.
- **가벼운 단일 실행 파일**: 별도의 설치 과정 없이 `.exe` 파일 더블클릭만으로 즉시 작동합니다.

## 3. 실행 방법
1. 제공받은 `get_programs.exe` 파일 (또는 `dist/get_programs.exe`)을 확인합니다.
2. 해당 실행 파일을 **더블 클릭**합니다.
3. 검은색 터미널(명령 프롬프트) 창이 열리며, 컴퓨터에 설치된 프로그램 목록과 총개수가 화면에 출력됩니다.
4. 내용을 모두 확인한 후 **엔터(Enter) 키**를 누르면 프로그램이 정상적으로 종료됩니다.
